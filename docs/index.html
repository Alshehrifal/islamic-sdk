Design Principles

Built for accuracy

Islamic dates are sensitive. This SDK is built around honesty — it never presents a calculated date as an official announcement.

01

Hilal-Aware

Every month-start date searches for official country announcements before falling back to Umm al-Qura calculation.

02

Transparent Status

Every date includes date_status: either "announced" with a source URL, or "calculated (unconfirmed)".

03

Arafah = Saudi Only

Yawm Arafah always follows Saudi Arabia's Dhul Hijjah. Hajj is in Mecca — this is not negotiable.

04

DRY Architecture

All API calls, Hijri math, and news scraping live in shared/. Skills are thin wrappers over solid utilities.
